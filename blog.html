1. What is DOM and what is the purpose of DOM?

Ans: The full form of DOM is the Document Object Model. It defines the logical structure of documents and the way a document is accessed and manipulated. DOM allows accessing and updating content in HTML.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: There are five rules for selecting using the HTML element DOM. Out of these two are the most commonly used. 1.getElementsByClassName() 2. getElementById(). To use an HTML element, use an ID in that HTML tag and declare that ID. Like this: getElementById(elementId)

DOM methods:

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Ans: The event bubble is, suppose a div contains many divs in sequence. eg: div>div>div. When the inner div is got by the function, all the outer divs get along with the inner div.

4. What is a callback function and why will you use it?

Ans: When a function is passed as an argument to another function it is called a callback function.
Instead of doing the same thing over and over again, the same thing can be done many times by putting it in a function and calling that function. This reduces the amount of code writing and takes less time.
